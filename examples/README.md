# Examples

Small runnable examples for each helper class.

Run any example from the package root:

```bash
php examples/arr.php
php examples/variable.php
```

Each script requires `vendor/autoload.php`, prepares its own demo data, and prints results with `print_r()` or `var_export()`.

| File | Class |
| --- | --- |
| `arr.php` | `Edrard\Helpers\Arr` |
| `bit.php` | `Edrard\Helpers\Bit` |
| `cli.php` | `Edrard\Helpers\Cli` |
| `date.php` | `Edrard\Helpers\Date` |
| `error.php` | `Edrard\Helpers\Error` |
| `form.php` | `Edrard\Helpers\Form` |
| `json.php` | `Edrard\Helpers\Json` |
| `obj.php` | `Edrard\Helpers\Obj` |
| `shell.php` | `Edrard\Helpers\Shell` |
| `str.php` | `Edrard\Helpers\Str` |
| `url.php` | `Edrard\Helpers\Url` |
| `variable.php` | `Edrard\Helpers\Variable` |
